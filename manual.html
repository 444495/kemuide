KEMU-IDE Instruction Manual

はじめに

KEMU-IDEは， The OpenWaseda Project の成果物です．内容はすべて MITライセンスの下， GitHub で公開されています．

このKEMU-IDEは，教育用マイコン KUE-CHIP2の動作をエミュレートすると共に， KUE-CHIP2の命令バイナリを出力するアセンブラを統合的に提供します．

アセンブラの操作

アセンブラ開発環境を起動

操作方法の説明

「新規作成」
現在の内容を消去して，新しいプログラムを作成します，
「開く」→「ファイルを開く」
アセンブラテキストのファイルをアップロードできます．
「開く」→「サンプルを開く」
サンプルプログラムを開きます．（未実装です．）
「アセンブル」
プログラムをアセンブルします． アセンブルとは，プログラムを人間可読な言語から， 機械可読な言語（機械語）に変換することです． 下の「出力」エリアに，アセンブル結果と， エラーメッセージ（失敗した場合）もしくは機械語列 （成功した場合）が出力されます．
「KUE-CHIP2で実行」
コンパイル結果が読み込まれた KUE-CHIP2エミュレータを起動します． エミュレータ上で「実行/停止」することで， プログラムを動作させることができます．
「マニュアル」
この説明書を開きます．
